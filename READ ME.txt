Välkommen till STEELMEET

Detta program är avsatt för den som vill genomföra sina styrkelyfts / enbart bänkpress tävlingar utan att behöva koppla upp sig till internet och som vill köra ett svenskt program som inte är norskt.

Jag har självmant utvecklat STEELMEET eftersom jag tyckte det inte fanns något program som var enkelt att använda samt gjord til den svenska marknaden. Det finns också bra alternativ som kostar pengar men de är flesta som har jag har påfunnit är avsädda för de feta amerikanerna. Därför tänkte jag det var lika bra att göra något som kan förbättra upplevelsen för lyftare samt klubbar. Ett bra program som alla kan köra ( sålänge man kör Windows ). Jag tycker inte man ska behöva ha en person "som kan programmet" utan vem som helst som har ett iq > rullstol ska kunna lära sig det. Många sena kvällar eller tidiga mornar beroende på hur man ser på saken, har gått in i STEEMEET. Ha så kul med testandet eller tävlingen eller vad du nu ska göra.

Donera om du / ni vill, det är motiverande. Det kommer gå till knark så jag kan arbeta mera.
